# MenuV2
This new version handles both the LCD and the keypad
```
const String menuItems = 
"-READ PINS:000"
"--SENSORS:000"
"---SENSOR A1:101"
"---SENSOR A2:102"
"--SWITCHES:000"
"---SWITCH PIN 24:103"
"---SWITCH PIN 26:104"
"-MOTOR:000"
"--START:105"
"--STOP:106";
Menu menu(menuItems); //Set up menu
```
That's it, you're done!
